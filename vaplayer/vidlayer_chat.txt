GOAL
----------------------------------------------------------------
Extract M3U8 / HLS stream URLs from:
  https://vaplayer.ru/embed/movie/tt2948356
Without using a browser if possible.

----------------------------------------------------------------
WHAT WAS BUILT
----------------------------------------------------------------

1. vidlayer_network_analyser.py
   - Browser-based extractor using selenium + Firefox headless
   - Injects JS interceptor into every frame to hook:
       fetch(), XHR, WebSocket, setAttribute,
       HLS.js, JWPlayer, Video.js, Clappr, Plyr, MediaSource
   - Captures Origin, Referer, and all request headers per URL
   - Visits each iframe URL directly
   - Parses every m3u8 found (master + variants + segments)
   - Stores headers needed to replay each stream
   - Usage:
       python vidlayer_network_analyser.py --url "https://vaplayer.ru/embed/movie/tt2948356" --wait 8
       python vidlayer_network_analyser.py --show-browser

2. direct_testing.py
   - Pure Python, no browser, no selenium
   - Calls the discovered API directly
   - Parses master/media playlists with stdlib only
   - Outputs VLC and ffmpeg ready-to-use commands
   - Usage:
       python direct_testing.py --imdb tt2948356 --type movie
       python direct_testing.py --imdb tt0944947 --type tv --season 1 --episode 1

----------------------------------------------------------------
KEY FINDINGS
----------------------------------------------------------------

Site structure:
  vaplayer.ru/embed/movie/tt2948356
    └── iframe: brightpathsignals.com/embed/movie/tt2948356
          └── player.min.js calls streamdata.vaplayer.ru/api.php
                └── returns stream_urls[] with m3u8 URLs
                      └── served from dataanalyticsacademy.site (CDN)

Real stream API:
  https://streamdata.vaplayer.ru/api.php
  Params: imdb, type, season (TV), episode (TV)

Required headers to use the API:
  Referer: https://brightpathsignals.com/embed/movie/
  Origin:  https://brightpathsignals.com

Required headers to replay streams:
  Referer: https://brightpathsignals.com/embed/movie/
